# AskFM Project
This is a simple project built using object-oriented programming (OOP) concepts in C++.
## The project, based on OOP concepts, allows you to perform the following actions:
1-Add questions
2-Add answers
3-Add a question to a previous answer
4-Add users
5-Display users
6-Delete questions
